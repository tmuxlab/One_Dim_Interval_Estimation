# One Dim Interval Estimationについて

製作者: Kento Oiwa

**One Dim Interval Estimation**は，ソフトウェアフォールト発見数データから，離散型非同次ポアソン過程における強度関数の推定値を元にブートストラップ方法を利用して区間推定を実行するプログラムです．使用するにあたって松井氏のrwseのリポジトリをcloneして併せて実行してください．

# One Dim Interval Estimationの使い方(区間推定部分)
## main_wse.R・main_tipshの使い方
このリポジトリをcloneした後, 必要なパッケージをインストールして使用してください.

## main2.Rの使い方
main_wse.Rもしくはmain_tipsh.Rの実行を行った後，BSsim_r.Rにて使用するNHPPシミュレーション方法を選択したのち使用してください．

## main3_r.Rの使い方
main2.Rの実行を行った後，使用してください．

詳しい使い方はmain1.R,main2.R,main3.Rをご覧ください．

# Interval Estimation using BSで使える各関数・ファイルについて
## PE()
生成されたサンプルm=10000本に点推定をかけるプログラム．
## IE()
点推定されたm=10000本から区間推定値を取得するプログラム．
## BS_Intervals_r.R
PEとIEの区間推定とそのほか記録コードをまとめたプログラム．メイン３における実行プログラム。
## BSsim_r.R
使用するNHPPシミュレーション方法をまとめた実行プログラム．使用する方法をここで選択しておく。
## SimMethodPathSource_r.R
NHPPシミュレーション方法のパスをまとめたプログラム。

#simMethod フォルダ内部のプログラムについて
## COS()
パラメトリック手法のNHPPシミュレーションプログラムの一つ．
逆確率変換法を用いてNHPP生成するプログラム．NHPPシミュレーションの一般的な問題を非一様確率変数生成の一般的な問題を変換している．
## GS-()
パラメトリック手法のNHPPシミュレーションプログラムの一つ．
指数変量の基本パッケージを使い，確率変量のオーダリングを必要としない．
## GS+()
パラメトリック手法のNHPPシミュレーションプログラムの一つ．
時間反転法を用いてGS-と同様にシミュレートを行うプログラム．
## RGS()
パラメトリック手法のNHPPシミュレーションプログラムの一つ．
パラメータ設定により一定条件下での実行は可能ですが，本プログラムでのループ実行は非推奨．
## TimTra()
ノンパラメトリック手法のNHPPシミュレーションプログラムの一つ．
実行時間が1日単位になりかねないので実行する際には注意してください．
## IntGen()
ノンパラメトリック手法のNHPPシミュレーションプログラムの一つ．
以前は実行不可であったものの、今では可能になっている。
## OrdSta()
ノンパラメトリック手法のNHPPシミュレーションプログラムの一つ．
DSなどの条件を変更すると実行不可になる可能性あり．
## ThiArr()
ノンパラメトリック手法のNHPPシミュレーションプログラムの一つ．
一般的に使用されている間引き法と呼ばれるサンプル生成プログラム．
## ThiInt()
ノンパラメトリック手法のNHPPシミュレーションプログラムの一つ．
間引き法に少し変更点を加えたようなサンプル生成プログラム．

# One Dim Interval Estimationの使い方(補正・拡張部分)
## IEBCa_log.R
点推定されたm=10000本から区間推定値のBCa補正値を取得するプログラム．
## collect_bca_coverage_grid.R
ヒートマップ保存用プログラム．
## compare_BCa_vs_others.R
すでに出力したファイルを元に，BCaとPerc/tQuickの比較を行うための関数を記述したプログラム．
強度関数の推定値と3種の補正手法の信頼区間を重ね書き（可視化）を行う．
## run_PEIE_addons_generic.R
WSE変換の一定条件下でのPerc/tQuick/BandSup等の補正結果を出力できる関数を記述したプログラム．
## run_PEIE_full_auto_plus.R
run_PEIE_addons_genericのフルオート拡張版を記述したプログラム．
## gen_viz_full_auto_plus.R
PIT系×Coverage系のマルチパネル図やサマリーのcsvファイルを取得するプログラム．
## coverage.R
compare_BCa_vs_othersの実行ファイル．
